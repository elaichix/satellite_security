# Reception Log — Dhaka Ground Station

## Log Format

Each reception session should be documented with the following fields:

| Field | Description |
|-------|-------------|
| Date | YYYY-MM-DD |
| Time (UTC) | HH:MM UTC |
| Time (BST) | HH:MM BST (UTC+6) |
| Satellite | Name and NORAD ID |
| Frequency | MHz |
| Max Elevation | Degrees |
| Duration | Minutes |
| Equipment | SDR model + antenna |
| Software | Decoder used |
| Quality | 1-5 rating |
| Image File | Path to saved image |
| Notes | Weather, interference, etc. |
